## Namespace: "ui/enums".TextTransform
Specifies different text transforms.

##### Variables
 - **none** - _String_.    
  No transform.
 - **capitalize** - _String_.    
  Text transform capitalize.
 - **uppercase** - _String_.    
  Text transform uppercase.
 - **lowercase** - _String_.    
  Text transform lowercase.
