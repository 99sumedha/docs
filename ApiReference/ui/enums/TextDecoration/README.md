## Namespace: "ui/enums".TextDecoration
Specifies different text decorations.

##### Variables
 - **none** - _String_.    
  No decoration.
 - **underline** - _String_.    
  Text decoration underline.
 - **lineThrough** - _String_.    
  Text decoration line-through.
