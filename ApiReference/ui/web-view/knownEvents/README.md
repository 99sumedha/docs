## Namespace: "ui/web-view".knownEvents
Known event names.

##### Variables
 - **loadFinished** - _String_.    
  Raised when the web-view has completely loaded an url.
 - **loadStarted** - _String_.    
  Raised when the web-view starts loading an url.
