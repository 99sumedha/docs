# Border
Using borders requires the "ui/border" module.
``` JavaScript
var borderModule = require("ui/border");
```
### Declaring a Border.
``` XML
 <Page>
     <Border borderRadius="10" borderWidth="1" borderColor="#FF0000">
         <Button text="OK"/>
     </Border>
 </Page>
```
