## Namespace: "connectivity".connectionType
Defines the different connection types.

##### Variables
 - **none** - _Number_.    
  Denotes no connection.
 - **wifi** - _Number_.    
  Denotes a WiFi connection.
 - **mobile** - _Number_.    
  Denotes a mobile connection, i.e. cellular network or WAN
